# Multi-unit monthly example data

A dataset containing simulated metric data for package examples. This
example is for two metrics with different units and data in monthly
format.

## Usage

``` r
multi_unit_monthly
```

## Format

A data frame with 132 rows and 5 variables:

- date:

  A vector of months.

- temp_a:

  An vector of random numbers.

- salinity_a:

  A vector of random numbers.

- temp_b:

  An vector of random numbers.

- salinity_b:

  A vector of random numbers.

## Source

Simulated data
