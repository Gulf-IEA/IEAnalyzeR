# Single metric formatted example data

A list object containing simulated metric data for package examples.
This example is for a single metric (biomass of red fish). This has been
prepped using the data_prep function, for use with IEAnalyzeR's plotting
function.

## Usage

``` r
single_data_formatted
```

## Format

A list obect of several dataframes:

- year:

  A vector of years.

- red_fish:

  A vector of random numbers.

## Source

Simulated data
